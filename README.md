# 🎓 EduConnect

EduConnect est une plateforme éducative sénégalaise moderne qui permet de connecter les étudiants avec des mentors et des enseignants afin de faciliter l’apprentissage, le suivi scolaire et la réussite académique.

---

## 🚀 Objectif du projet

Ce projet a été réalisé dans un cadre scolaire afin de :
- Créer un site web moderne et responsive
- Mettre en pratique HTML et CSS
- Simuler une vraie plateforme éducative
- Améliorer l’expérience utilisateur

---

## 🌐 Aperçu du site

👉 Page d’accueil avec sections :
- Hero (bannière principale)
- À propos
- Services
- Mentors
- Témoignages
- Contact

---

## ✨ Fonctionnalités

- 📚 Services éducatifs
- ⭐ Témoignages utilisateurs
- 📩 Formulaire de contact
- 📱 Site responsive (mobile / desktop)
